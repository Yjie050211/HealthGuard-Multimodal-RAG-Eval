# CareEvent-VLM-RAG-Eval

## 项目简介

本项目面向智能照护与居家健康监护场景，围绕跌倒、长时间静止、异常徘徊、遮挡误判等典型异常事件，构建一个轻量级的多模态异常事件理解与可信问答评测原型系统。

项目基于国家级大创“中云智护”的应用背景，当前重点聚焦算法侧预研，不依赖深度相机、激光雷达或机器人实物硬件，主要通过公开图像/视频样例、模拟事件数据和照护知识库完成多模态理解、RAG 问答与评测流程验证。

## 研究问题

1. 传统行为识别模型只能输出“是否异常”，如何进一步生成可解释的异常事件描述？
2. 多模态大模型在智能照护场景下是否容易产生错误解释或不可靠建议？
3. RAG / GraphRAG 思路能否提升照护建议的可追溯性与可信度？
4. 如何构建一个轻量级评测流程，对比传统模型、纯 VLM、VLM + RAG 的差异？

## 技术路线

* 异常事件建模：构建跌倒、静止、徘徊、遮挡等典型场景样本。
* 多模态理解：使用 Qwen-VL / VideoLLaMA 等多模态模型进行异常事件语义解释。
* 知识增强问答：构建照护知识库，引入 RAG 检索生成，减少大模型凭空生成建议。
* 评测分析：从异常判断、解释一致性、可追溯性、风险等级判断、误报分析等维度进行比较。
