# HealthGuard-Multimodal-RAG-Eval

## 项目简介

本项目面向居家养老与远程健康监护场景，构建一个轻量级的多模态异常事件理解与可信问答评测原型系统。

项目基于国家级大创“中云智护”的应用背景，当前重点聚焦算法侧预研，不依赖深度相机、激光雷达或机器人实物硬件，而是通过公开图像/视频样例、模拟事件数据和健康监护知识库，完成多模态异常事件理解、RAG 知识增强问答和评测分析流程验证。

## 研究目标

传统健康监护系统往往只能输出“是否异常”，但在真实照护场景中，系统还需要回答：

1. 发生了什么异常？
2. 为什么判断为异常？
3. 风险等级如何？
4. 是否需要报警或二次确认？
5. 处理建议是否有知识依据？
6. 多模态大模型是否存在幻觉或过度判断？

本项目尝试将异常行为识别、多模态大模型、RAG / GraphRAG 思路和垂直场景评测结合，探索健康监护场景下可解释、可追溯的决策支持方法。

## 技术路线

```text
公开图像/视频样例 + 模拟事件数据
        ↓
异常事件样本构建与结构化标注
        ↓
传统 baseline / 多模态大模型
        ↓
异常类型、风险等级、视觉证据、处理建议
        ↓
健康监护知识库 + RAG 检索增强
        ↓
可信问答与可追溯建议
        ↓
对比实验与误差分析
