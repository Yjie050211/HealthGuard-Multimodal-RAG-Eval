# 初步对比实验设计

| 方法                      | 异常判断 | 解释能力 | 可追溯性 | 误报分析 | 实现成本 |
| ------------------------- | -------- | -------- | -------- | -------- | -------- |
| Rule Baseline             | 中       | 弱       | 无       | 弱       | 低       |
| ST-GCN / SVM Baseline     | 中       | 弱       | 无       | 中       | 中       |
| Pure VLM                  | 中高     | 中       | 无       | 中       | 中       |
| VLM + Simple RAG          | 中高     | 较强     | 中       | 较强     | 中       |
| VLM + GraphRAG / LightRAG | 预期较强 | 强       | 强       | 强       | 中高     |

## 当前结论

传统分类模型适合完成“是否异常”的判别，但难以解释异常原因和给出可追溯建议。纯多模态大模型具备较强语义解释能力，但存在幻觉和过度判断风险。引入 RAG 后，模型能够基于照护知识库生成更可追溯的建议，后续将进一步探索图增强 RAG 在多跳风险分析中的作用。
